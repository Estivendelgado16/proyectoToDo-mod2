# ToDo App - App de Gestión de Tareas

## Descripción / Description
**App de Tareas** es una aplicación web simple y eficiente para gestionar tus tareas diarias. Te permite crear, editar, eliminar y filtrar tareas con persistencia en el navegador usando LocalStorage. ¡Organiza tu día y sé más productivo!

**ToDo App** is a simple and efficient web app for managing daily tasks. Create, edit, delete, and filter tasks with browser LocalStorage persistence. Organize your day and be more productive!

## Características / Features
- ✅ Crear tareas con título, categoría (ej. Casa, Trabajo) y prioridad (Alta, Media, Baja).
- ✏️ Editar tareas (título, categoría, prioridad).
- 🗑️ Eliminar tarea individual (botón X) o borrar completadas en masa.
- 🔍 Filtros: Todas, Pendientes, Completadas.
- 📊 Contador de tareas pendientes.
- 💾 Persistencia automática con LocalStorage (no se pierden al recargar).
- 📱 Responsive design (móvil-friendly).
- 🎨 Icons de Font Awesome.

## Tecnologías / Tech Stack

## Instalación Rápida / Quick Start
1. Clona o descarga el repositorio:  
   `git clone https://github.com/estivedelgado16/proyectoToDo-mod2.git`
2. Abre `index.html` en tu navegador (Chrome, Firefox, etc.).  
   **No necesitas servidor!** Funciona 100% local.
3. (Opcional) Usa VS Code + Live Server extension para hot reload.

```bash
# Windows
start index.html

# macOS/Linux
open index.html  # o xdg-open index.html
```

## Uso / Usage
1. **Agregar tarea**: Escribe en \"¿Qué necesita ser hecho?\" y presiona Enter.
2. **Marcar completada**: Checkbox al lado de la tarea.
3. **Editar**: Click en ícono lápiz (✏️), modifica y guarda.
4. **Eliminar**: Click en X o \"Borrar completados\".
5. **Filtrar**: Botones en footer (Todas / Pendientes / Completadas).
